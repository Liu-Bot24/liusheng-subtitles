语音识别 API 指南

语音识别 API 用来把视频声音转成原文字幕。设置完成后，回到“任务与字幕”页选择原音频语言并开始抽取。

在线 API

插件内置以下平台 ASR 配置档案，选择对应档案模板填入 API Key 即可直接使用。也可自行添加 Whisper 兼容接口档案，需要填写接口地址、模型名称和 API Key。

阿里云百炼 Fun-ASR

  1. 在 百炼控制台创建 API Key。
  2. 在插件“语音识别”里选择“Fun-ASR”。
  3. 把 API Key 填入“API 密钥”。
  4. 保存设置。

价格：约 0.0132 元/分钟。新用户免费额度和有效期以官方页面为准。

Groq Whisper

  1. 在 Groq 控制台创建 API Key。
  2. 在插件“语音识别”里选择“Groq Whisper”。
  3. 把 API Key 填入“API 密钥”。
  4. 保存设置。

价格：约 0.00067 美元/分钟（0.04 美元/小时）。

OpenAI Whisper

  1. 在 OpenAI 平台创建 API Key。
  2. 在插件“语音识别”里选择“OpenAI Whisper”。
  3. 把 API Key 填入“API 密钥”。
  4. 保存设置。

价格：0.006 美元/分钟。

xAI Grok

  1. 在 xAI 控制台创建 API Key。
  2. 在插件“语音识别”里选择“xAI Grok”。
  3. 把 API Key 填入“API 密钥”。
  4. 保存设置。

价格：约 0.00167 美元/分钟（0.10 美元/小时）。

以上价格按 2026 年 5 月公开页面换算，可能发生变动；实际费用以各平台控制台和账单为准。

本地部署

本地 Speaches 启动后，插件接口地址填写 http://127.0.0.1:8000/v1。

Windows

  1. 安装 Docker Desktop。
  2. 安装或更新 NVIDIA 驱动。
  3. 打开 Docker Desktop，等待 Docker 开始运行。
  4. 打开 PowerShell。
  5. 执行下面的命令。
docker rm -f speaches
docker run --detach `
  --publish 8000:8000 `
  --name speaches `
  --volume speaches-hf-cache:/home/ubuntu/.cache/huggingface/hub `
  --gpus=all `
  ghcr.io/speaches-ai/speaches:latest-cuda
  1. 浏览器打开 http://127.0.0.1:8000/docs，能看到 Speaches 页面后继续。
  2. 识别字幕时保持 Docker Desktop 和 speaches 容器运行。

Mac

  1. 安装 Docker Desktop。
  2. 打开 Docker Desktop，等待 Docker 开始运行。
  3. 打开终端。
  4. 执行下面的命令。
docker rm -f speaches 2>/dev/null || true
docker run --detach \
  --publish 8000:8000 \
  --name speaches \
  --volume speaches-hf-cache:/home/ubuntu/.cache/huggingface/hub \
  ghcr.io/speaches-ai/speaches:latest-cpu
  1. 浏览器打开 http://127.0.0.1:8000/docs，能看到 Speaches 页面后继续。
  2. 识别字幕时保持 Docker Desktop 和 speaches 容器运行。

插件设置

正在编辑 选择“自定义档案”，或点击“新增档案”。
档案名称 本机 Speaches ASR
接口类型 Whisper 兼容接口
接口地址 http://127.0.0.1:8000/v1
模型名称 Systran/faster-whisper-large-v3
VAD 过滤 自动
API 密钥 local
保存后 回到“任务与字幕”，选择原音频语言，然后开始抽取。

注意事项

  • 第一次启动 Speaches 时会下载模型，等待时间会更长。
  • 再次使用会复用已经下载的模型。
  • 识别时不要关闭 Docker Desktop。
  • 字幕语种不对时，先选择正确的原音频语言，再重新 ASR。
