翻译模型配置说明

翻译模型 API 用来把原文字幕翻译成目标语言。设置完成后，回到“任务与字幕”页开始抽取，或对已有原文重新翻译。

内置模板

插件内置硅基流动、阿里云百炼、火山引擎和 OpenRouter 档案。选择对应模板并填入 API Key 即可使用；默认模型已填好，也可以在“模型名称”中选择或输入平台支持的模型 ID。

硅基流动

  1. 在硅基流动控制台创建 API Key。
  2. 在插件“翻译模型”里选择“硅基流动”。
  3. 把 API Key 填入“API 密钥”。
  4. 需要更换模型时，在“模型名称”里选择或填写模型 ID。
  5. 保存设置。

阿里云百炼

  1. 在百炼控制台创建 API Key。
  2. 在插件“翻译模型”里选择“阿里云百炼”。
  3. 把 API Key 填入“API 密钥”。
  4. 需要更换模型时，在“模型名称”里选择或填写模型 ID。
  5. 保存设置。

火山引擎

  1. 在火山引擎方舟控制台创建 API Key。
  2. 在插件“翻译模型”里选择“火山引擎”。
  3. 把 API Key 填入“API 密钥”。
  4. 需要更换模型时，在“模型名称”里选择或填写模型 ID。
  5. 保存设置。

OpenRouter

  1. 在 OpenRouter 创建 API Key。
  2. 在插件“翻译模型”里选择“OpenRouter”。
  3. 把 API Key 填入“API 密钥”。
  4. 需要更换模型时，在“模型名称”里选择或填写模型 ID。
  5. 保存设置。

以上均有新用户免费额度或免费模型，有效期以官方页面为准。

自定义档案

自定义档案用于接入其他翻译模型服务。新增档案后，按服务商文档填写接口格式、接口地址、模型名称和 API Key。LM Studio、Ollama 或其他 llama 本地模型工具只要提供兼容接口，也可以用自定义档案接入。

档案名称 写一个自己能认出来的名称，只影响档案列表显示。
接口格式 接入 Chat Completions 风格接口时选择“OpenAI 兼容格式”；接入 Anthropic Messages 风格接口时选择“Anthropic 兼容格式”。
接口地址 填写服务商提供的 API Base URL。本机模型服务也填写它自己的本机地址，例如 http://127.0.0.1:1234/v1 或 http://127.0.0.1:11434/v1。
模型名称 填写平台控制台、文档或本地模型工具里显示的模型 ID。
API 密钥 填写该服务创建的 API Key；本地模型工具如果不校验密钥，可以填写任意占位文本。
目标语言 选择字幕要翻译成的语言。保存后，新任务和“重新翻译字幕”都会使用当前目标语言。
翻译并发 控制同时请求翻译模型的字幕分组数量。遇到限流或返回不稳定时调低。
保存后 新任务会使用当前翻译档案；已有原文字幕可以点击“重新翻译字幕”。

注意事项

  • 翻译模型需要稳定按插件要求返回结构化结果；部分模型的内容审查可能较严格，如果经常失败，可以更换模型或降低翻译并发。
  • “重新翻译字幕”只会使用已有原文重新调用翻译模型，不会重新调用 ASR。
  • 如果使用本机模型服务，翻译字幕时保持该服务运行。
